# todolist1

Language: HTML / CSS / Javascript

Build Tool: Parcel

구현 기능: 
  1. 새 메모 입력
  2. 기존 메모 삭제
  3. 기존 메모 전체 삭제
  4. 메모 체크 박스 선택시 스타일 효과
  5. 메모 전체 선택
  6. 로컬스토리지에서 메모 데이터 저장하기 및 불러오기
  7. 로컬스토리지에서 데이터 불러올 때 기존에 적용하였던 체크박스
